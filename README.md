EvanCard是一个超极简而精致的仿编程页面的在线信息卡片，可以放置任何你想放入的信息。
我现在把它当成一个Business Card在使用，您也可以放一些其他信息或文字。

### 特色
- 页面以深色/浅色双主题为视觉核心
- 卡片式编程界面布局让信息一目了然
- 除信息外还可以自行编辑其他功能
